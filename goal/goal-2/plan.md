## 风险

- 直接抓取或声称使用真实瑞幸会员信息会有隐私和合规风险。
- 公开菜单、价格、优惠会随时间变化，必须避免写死为“官方实时”。
- 商品图片有版权和授权风险，不能随意复制第三方素材到项目中；若引用远程公开图片，要注明来源与替代方案。
- 浏览器地理位置涉及用户敏感权限，必须提供拒绝授权 fallback，并避免持久化精确坐标。
- 自动点单模拟不能暗示真的操作用户手机或真实瑞幸 App；必须清晰标注为本地 UI 演示，并在支付页停止。
- UI 加料过多会导致移动端拥挤。
- 决策逻辑复杂化后可能出现类型不一致或推荐解释对不上。
