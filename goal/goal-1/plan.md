# Goal 1 Plan: 瑞幸 AI Agent 流量运营方案报名交付

## 需求理解

用户准备参加飞书活动页面中的瑞幸咖啡命题，需要完成报名表内容、开题报告、补充材料、可运行 demo，并尽可能将 demo 放到用户 GitHub，最终辅助完成报名提交。截图显示至少需要填写：

- 开题报告 part 1：命题前置分析与洞察，50-300 字。
- 开题报告 part 2：整体解决方案设计，300-600 字。
- 可选补充材料：附件和链接，可包含开题补充材料、项目经验、数据分析样本、研究笔记、参考资料清单、GitHub/个人主页/Notion 链接等。

用户希望参考 `https://github.com/bcefghj?tab=repositories` 的项目风格，重点不是只写文本，而是产出一个可演示、可提交、可托管到 GitHub 的完整方案。

## 上下文

- 当前不是已有代码项目，因此新建项目目录：`luckin-agent-submission/`。
- 活动页为动态页面，普通抓取不稳定，需要通过浏览器或用户登录状态才能最终提交。
- 用户提供了报名表截图，可以直接确定核心文本字段要求。
- GitHub 推送依赖本机 `gh` 或 `git` 的认证状态，以及用户账号/仓库权限。

## 风险

- 活动页面内容、报名截止时间、表单字段可能变化，需要联网或浏览器再次确认。
- 最终报名提交可能需要飞书/活动账号登录、个人信息、队伍信息或验证码，无法无授权代填敏感信息。
- 将项目推到 GitHub 需要用户本地认证；若未登录，只能准备仓库内容与推送命令。
- 不能照搬 bcefghj 或其他开源项目代码；只能学习交付结构、README 叙事、demo 展示方式。
- 瑞幸真实业务数据不可得，需要用合成数据和明确标注的假设进行 demo。
- Agent demo 若接入真实 LLM API，需要用户提供密钥；本次优先做无密钥可运行的规则 + 检索 + 模拟 Agent，保证可演示。

## 执行方案

1. 调研活动要求、瑞幸业务背景、AI Agent 运营相关公开资料、bcefghj 参考仓库。
2. 形成参赛方案定位：围绕“品牌无法直触用户背景下的意图洞察、会员关系重构、唤醒、推荐、下单、复购、忠诚闭环”。
3. 产出报名文本：part 1 和 part 2，控制在截图建议字数内。
4. 产出补充材料：README、方案白皮书、架构说明、研究笔记、数据样例、评委可快速理解的提交摘要。
5. 构建 demo：一个可本地运行的 Luckin Agent 流量运营工作台，至少包含用户意图识别、会员分层、动作推荐、唤醒话术、下单路径建议、复购闭环指标。
6. 验证 demo：安装依赖、运行构建、浏览器截图、关键流程手测。
7. 初始化 Git 仓库并提交；如 GitHub 已认证，创建/推送远程仓库；否则留下可执行推送说明。
8. 尝试打开活动报名页面，辅助填写或提供可复制内容；若遇到登录/验证码/个人隐私字段，则明确需要用户接管。
9. 最终全面 review：C 端体验、代码质量、安全性、材料完整性、提交可用性。

## 验证方式

- 文案验证：字数符合要求，内容覆盖截图中指定点，避免空泛口号。
- 资料验证：补充材料路径完整，引用来源可追踪，假设与合成数据有标注。
- 代码验证：`npm install`、`npm run build`、本地 dev server 可用。
- UI 验证：Playwright 打开页面、截图、检查核心交互无报错。
- Git 验证：本地 `git status` 干净，提交存在；如可推送，远程链接可访问。
- 报名验证：生成可粘贴字段、附件/链接清单；如能进入表单，则尽量填充并在最终提交前等待用户确认。

## 回滚方案

- 所有变更限定在 `luckin-agent-submission/` 与必要输出目录内。
- 若 demo 技术方案复杂度过高，回退为静态 Vite/React 单页 demo，保证可运行和可展示。
- 若 GitHub 推送失败，保留本地 Git 仓库、完整提交和推送命令，不删除本地成果。
- 若活动表单无法访问或需要登录，回退为提交包和逐项粘贴指南，由用户完成最后授权动作。
- 若外部资料无法稳定访问，使用已保存的 research notes 和来源链接作为补充材料，并标注访问日期。

## 完成标准

- `goal/goal-1/input.md`、`plan.md`、`tasks.md` 建立并持续更新。
- 可运行 demo 和材料包完成。
- 报名表两段文字完成并可直接粘贴。
- 补充材料附件与链接准备完成。
- Git 本地提交完成；尽可能推送到 GitHub。
- 活动提交流程推进到可由用户确认提交或明确说明唯一阻塞点。
